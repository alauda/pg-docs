# Примечания к выпуску

## v4.0.0

### Новые и оптимизированные функции

#### Разделение компонента и платформы

Поддержка Alauda для жизненного цикла PostgreSQL была декомпозирована от поддержки платформы. Это означает, что когда клиенту нужна определенная функция или исправление ошибки, команда PostgreSQL может выпустить обновление немедленно, без ожидания выпуска платформы.

#### Логи в реальном времени на странице деталей компонента

Логи на странице деталей PostgreSQL были заменены логами в реальном времени, которые обновляются мгновенно, что облегчает устранение неполадок и анализ проблем.

#### Доступна новая версия

Поддержка PostgreSQL v14.0.

# Обновление

Поддержка Alauda для PostgreSQL будет выполнять обновления в соответствии с настроенной стратегией обновления:

- **Автоматическое** : Автообновления инициируются немедленно при обнаружении новых версий компонентов.
- **Ручное** : Требуется ручное одобрение перед началом процесса обновления.

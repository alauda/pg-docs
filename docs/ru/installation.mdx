# Установить

## Установка основных сервисов данных Alauda Container Platform

::: tip Опциональная установка
Плагин служит основным компонентом для представления сервисов данных. Установка не требуется, если **веб-консоль** <span /> не нужна.
:::

### Предварительные требования

1. Скачайте пакет установки основных сервисов данных Alauda Container Platform, соответствующий платформе.
2. Используйте возможности публикации приложений платформы для публикации основных сервисов данных Alauda Container Platform в кластер `global`.

### Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. В левом навигационном баре перейдите в **Marketplace** > **Плагины кластера**.
    2. Выберите кластер `global`.
    3. Нажмите кнопку действий рядом с плагином **Основные сервисы данных Alauda Container Platform** > **Установить**.
  </Tab>
</Tabs>

## Установка фреймворка RDS Alauda Container Platform

::: tip Опциональная установка
Плагин служит основным компонентом для представления сервисов данных. Установка не требуется, если для вас не нужна веб-консоль.
:::

### Предварительные требования

1. Скачайте пакет установки фреймворка RDS Alauda Container Platform, соответствующий платформе.
2. Используйте возможности публикации приложений платформы для публикации фреймворка RDS Alauda Container Platform в любой кластер, где необходимы сервисы данных.

### Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. Войдите в платформу и перейдите на страницу **Администратор**.
    2. В левом навигационном баре выберите **Marketplace** -> **Operator Hub**, чтобы перейти на страницу **Operator Hub**.
    3. Найдите **Фреймворк RDS Alauda Container Platform**, нажмите **Установить**, чтобы перейти на страницу установки.

    Параметры конфигурации:

    | **Параметр**             | **Рекомендуемая конфигурация**                                                                                                                   |
    | :------------------------ | :---------------------------------------------------------------------------------------------------------------------------------------------- |
    | **Канал**                | Канал по умолчанию — **stable**.                                                                                                              |
    | **Версия**               | Выберите желаемую версию для установки.                                                                                                          |
    | **Режим установки**       | **Кластер**: Все пространства имен в кластере используют один оператор для создания и управления экземплярами, что приводит к снижению использования ресурсов.        |
    | **Место установки**       | Выберите **Рекомендуемое**: если таковое отсутствует, оно будет создано автоматически.                                                                   |
    | **Стратегия обновления**  | **Ручная**: Когда новая версия доступна в Operator Hub, требуется вручную подтвердить обновление оператора до последней версии. |

    4. На странице **Установить оператора** выберите **Конфигурация по умолчанию**, затем нажмите **Установить**, чтобы завершить установку **Фреймворка RDS Alauda Container Platform**.
  </Tab>
</Tabs>

# 升级

:::warning
PostgreSQL 11 已弃用，将在未来版本中移除。请升级到 PostgreSQL 12、14 或 16。
:::

Alauda 对 PostgreSQL 的支持将根据配置的升级策略执行升级：

- **自动** : 一旦检测到新组件版本，系统会立即触发自动升级。
- **手动** : 在启动升级过程之前，需进行手动审核以获得批准。

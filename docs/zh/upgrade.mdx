# 升级

:::warning
PostgreSQL 11 已弃用，将在未来版本中移除。请升级到 PostgreSQL 12、14 或 16。
:::

灵雀云的 PostgreSQL 数据库服务将根据配置的升级策略执行升级：

- **自动** : 自动升级将在检测到新组件版本时立即触发。
- **手动** : 在启动升级过程之前，需要手动批准。

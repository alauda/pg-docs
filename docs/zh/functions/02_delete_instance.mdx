# 删除 PostgreSQL 实例

:::warning
**删除前重要注意事项**
- 该操作**不可逆** - 删除的实例无法恢复
- 如果后续可能需要数据，请确保有[有效备份](../functions/15_back_restore.mdx)
- 确认没有应用程序正在使用该数据库实例
- 集群删除将移除所有副本和相关资源
:::

删除 PostgreSQL 实例将移除：
- 自定义资源 (CR)
- StatefulSet 和 Pods
- ConfigMaps 和 Secrets
- 持久卷声明 (PVCs)
- 监控和日志资源

## 操作步骤

<Tabs>
  <Tab label="命令行操作">
    要删除 PostgreSQL 实例（包括所有关联的 PVC），执行：

    ```bash
    kubectl -n $NAMESPACE delete postgresql <实例名称>
    ```

    示例输出：
    ```bash
    postgresql.acid.zalan.do "example-postgres" deleted
    ```

    删除后，可以通过以下命令确认实例状态：

    ```bash
    kubectl -n $NAMESPACE get postgresql
    ```
  </Tab>

  <Tab label="Web 控制台">

    1. 在左侧菜单中导航到 **PostgreSQL**
    2. 选择目标命名空间
    3. 从实例列表中选择要删除的实例
    4. 在实例详情页面的右上角，点击**操作**按钮
    
    5. 选择**删除**选项
    6. 确认删除实例及其所有数据
    7. 在确认框中输入实例名称
    8. 点击**删除**
  </Tab>
</Tabs>

## 删除后验证

删除后，验证所有资源是否已移除：

```bash
kubectl -n default get postgresql  # 不应显示已删除的实例
kubectl -n $NAMESPACE get pvc  # 验证 PVC 是否已删除
```

# 备份恢复

## 功能介绍
提供物理备份和逻辑备份两种方式保障数据安全。

<Tab label="界面操作">

1. 在左侧导航栏中，单击 **PostgreSQL**
2. 选择目标命名空间
3. 在实例列表中选择要删除的实例
4. 点击**备份恢复**Tab页

### 创建备份
1. 如果未设置备份存储，请根据页面提示先**配置页面存储**。
2. 点击**创建备份**, 并按照提示完成备份创建。
### 数据库恢复
1. 点击**数据库恢复**按钮，按照提示完成数据库恢复操作, 将备份数据恢复到一个新数据库。
### 恢复记录及归档列表
1. 点击**恢复记录**和**归档列表**查看相关信息。

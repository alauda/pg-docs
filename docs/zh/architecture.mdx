# 架构

请增加一张或多张关于该功能的技术架构图，图中可以包含以下内容：

- 核心组件：例如 Operator、任务、中间件等，并在图中描绘出各个组件之间的联系及其作用。
- 数据流程：例如引入的资源、资源之间的依赖关系，以及资源被消费的流程等。
- 部署架构：例如多集群部署、高可用设计等。

在架构图下方增加对该图片的描述，说明图片的含义及其中各部分的作用。

对架构图中引入的概念进行简要介绍。如果某个概念需要更详细的说明，请链接到「核心概念」对应的章节。---
